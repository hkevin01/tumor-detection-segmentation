## âŒ Never Commit

### Medical Data
- `*.nii`, `*.nii.gz` - NIfTI files
- `*.dcm`, `*.dicom` - DICOM files
- `**/imagesTr/`, `**/labelsTr/` - Dataset directories
- `**/patient_data/` - Patient data
- `**/*_phi_*` - PHI patterns

### Large Files
- `*.pth`, `*.pt`, `*.ckpt` - Model weights
- `*.zip`, `*.tar.gz` - Archives
- `*.npy`, `*.npz` - NumPy arrays
- `*.h5`, `*.hdf5` - HDF5 files

### Datasets
- `**/BraTS*/` - BraTS datasets
- `**/Task*_*/` - MSD datasets
- `**/HECKTOR*/` - HECKTOR data
- `mlruns/` - MLflow artifacts
- `cache/` - Cache directories

## ğŸ”§ Quick Fixes

### If you accidentally staged a large file:
```bash
git reset HEAD path/to/large/file
```

### If you already committed a large file:
```bash
# Remove from history (use carefully!)
git filter-repo --path path/to/file --invert-paths
```

### If you need to track a large file:
```bash
# Use Git LFS instead
git lfs track "*.pth"
git add .gitattributes
```
